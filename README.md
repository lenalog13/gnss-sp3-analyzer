# GNSS SP3 Analyzer

Программный комплекс для оценки точности расчетных SP3-эфемерид ГНСС ГЛОНАСС при сопоставлении с эталонными данными.

---

## Назначение

Проект разработан в рамках дипломной работы:
"Разработка программного комплекса оценки точности расчетных SP3-эфемерид ГНСС ГЛОНАСС"

---

## Возможности

- Загрузка SP3 и CLK файлов
- Расчет ошибок координат (XYZ, RTN)
- Расчет статистических метрик:
  - RMS (X, Y, Z, 3D)
  - RMS (R, T, N)
  - Mean / Max ошибки
  - Clock RMS
- Визуализация результатов (графики)
- Сохранение результатов в PostgreSQL
- Просмотр истории экспериментов
- Повторная загрузка результатов

---

## Архитектура
    GUI (PySide6)
    ↓
    Controller (Python)
    ↓
    Database (PostgreSQL)


---

## Интерфейс

Приложение включает:

- Панель загрузки файлов
- Графики ошибок орбит и часов
- Таблицу значений по эпохам
- Панель статистики
- Окно истории экспериментов

---

## Технологии

- Python (PySide6, NumPy, pyqtgraph)
- C++ (планируется backend)
- PostgreSQL
- psycopg2
