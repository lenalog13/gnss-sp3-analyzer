# GNSS SP3 Analyzer

Программный комплекс для оценки точности SP3 эфемерид ГНСС ГЛОНАСС.

## Функции

- Загрузка SP3 и CLK файлов
- Расчет ошибок XYZ и RTN
- Визуализация орбитальных ошибок
- Статистический анализ

## Технологии

- Python (PySide6)
- C++ backend (планируется)
- PostgreSQL (планируется)
